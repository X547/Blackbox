CtlWord8

CtlWord8 contains the automation interface for Microsoft Word 8.0. The objects contained in this module are explained in the corresponding help file (VBAWRD8.HLP) which is located on the MS Office CD but is not installed by default.
This Module is equal to the module CtlWord in previous releases of BlackBox.
