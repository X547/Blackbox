CtlOutlook8

CtlOutlook8 contains the automation interface for Microsoft Outlook 8.0. The objects contained in this module are explained in the corresponding help file (VBAOUTL.HLP) which is located on the MS Office CD but is not installed by default.
This Module is equal to the module CtlOutlook in previous releases of BlackBox.
