CtlAccess8

CtlAccess8 contains the automation interface for Microsoft Access 8.0. The objects contained in this module are explained in the corresponding help file (ACVBA80.HLP) which is located on the MS Office CD but is not installed by default.
This Module is equal to the module CtlAccess in previous releases of BlackBox.
