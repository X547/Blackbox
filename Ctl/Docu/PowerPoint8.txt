CtlPowerPoint8

CtlPowerPoint8 contains the automation interface for Microsoft PowerPoint 8.0. The objects contained in this module are explained in the corresponding help file (VBAPPT8.HLP) which is located on the MS Office CD but is not installed by default.
This Module is equal to the module CtlPowerPoint in previous releases of BlackBox.
