CtlDAO35

CtlDAO35 contains the automation interface for Microsoft Data Access Objects 3.5 (DAO). The objects contained in this module are explained in the corresponding help file (DAO35.HLP) which is located on the MS Office CD.
This Module is equal to the module CtlDAO in previous releases of BlackBox.
