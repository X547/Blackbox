CtlExcel8


CtlExcel8 contains the automation interface for Microsoft Excel 8.0. The objects contained in this module are explained in the corresponding help file (VBAXL8.HLP) which is located on the MS Office CD but is not installed by default.
This Module is equal to the module CtlExcel in previous releases of BlackBox.
